# Pannello Allegati della gestione Conti

## Regole
### Finalità del pannello
Il pannello Allegati della gestione Conti è predisposto per allegare e associare a ciascun conto documenti esterni di supporto.

### Tipologie di file gestibili
Nel pannello possono essere associati diversi tipi di file, ad esempio PDF, Excel, Word e altri documenti analoghi.

### Origine dei file
Il caricamento dei file può essere eseguito sia da server sia da dispositivo locale.

### Funzioni disponibili
Nel pannello è prevista la visualizzazione della lista degli allegati associati al conto.

### Preview del documento
Oltre alla lista degli allegati, il pannello consente anche la preview del documento allegato.
