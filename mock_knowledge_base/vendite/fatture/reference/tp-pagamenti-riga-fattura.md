# TP Pagamenti della riga fattura

## Campi

### Condizioni di pagamento
Riporta le condizioni di pagamento impostate nella testata della fattura.

### Data scadenza
Può essere indicata se la soluzione di pagamento non è impostata.

### Tipo imputazione
Permette di indicare il tipo di imputazione:
- a percentuale
- a valore

## Regole

### Proposta dalla testata
Nel TP Pagamenti vengono proposte le condizioni di pagamento presenti nella testata della fattura.

### Modifica pagamenti
Le condizioni di pagamento proposte possono essere modificate sulla riga.
