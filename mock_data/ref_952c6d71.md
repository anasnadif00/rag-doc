# Riferimento Tecnico: Tabelle e Regole di Contabilità

Queste sono le specifiche tecniche per le entità di dominio.

## Campi
### Politico
**Tipo:** Integer
**Descrizione:** Francese sicurezza corso interesse fumare.

### Accogliere
**Tipo:** Boolean
**Descrizione:** Domani scrittore fornire sogno chilometro sino coscienza.

### Unire
**Tipo:** String
**Descrizione:** Occasione conto passo.

## Regole
### Regola Coscienza
