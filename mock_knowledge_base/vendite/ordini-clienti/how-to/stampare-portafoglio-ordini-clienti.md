# Stampare il portafoglio Ordini Clienti

## Prerequisiti
La stampa **Portafoglio Ordini Clienti** consente di ottenere un report riepilogativo degli ordini clienti e delle relative informazioni commerciali, logistiche e di avanzamento.


La stampa può essere prodotta in formato PDF oppure esportata in Excel per successive elaborazioni.

## Procedura
1. Aprire l'elaborazione **Portafoglio Ordini Clienti** dal modulo **Ordini**.

2. Se necessario, filtrare gli ordini per **Numero ordine**.

3. Se necessario, filtrare gli ordini per **Operatore**.

4. Se necessario, compilare l'intervallo **Da data** e **A data**.

5. Se necessario, filtrare per **Articolo**.

6. Se necessario, filtrare per **Cliente**.

7. Se necessario, filtrare in base alle **Quantità evase forzatamente**.

8. Se necessario, filtrare per **Descrizione articolo**.

9. Se necessario, filtrare per **Stato ordine**.

10. Se necessario, filtrare per **Zona**.

11. Se necessario, filtrare per **Nazione**.

12. Se necessario, filtrare per **Commessa**.

13. Impostare il criterio di **Ordinamento** scegliendo tra:
    - numero ordine;
    - cliente;
    - articolo;
    - ragione sociale.

14. Selezionare il formato di output:
    - PDF;
    - Excel.

15. Lanciare la stampa o l'estrazione
16. Questa stampa ci permette di fare una selezione, diciamo un report, degli ordini riepilogando una serie di informazioni, tra le quali:

- ovviamente cliente
- numero dell'ordine
- prodotto ordinato
- zona
- agente
- nazione  

ma anche di verificare:

- le quantità ordinate
- i valori ordini
- le quantità consegnate
- il valore consegnato
- eventuali quantità evase forzatamente
- lo stato dell'ordine, quindi se si trova in evasione, se è inserito o se è confermato ed eventuali quantità residue ancora da consegnare


## Verifiche finali
Controllare che il report riporti gli ordini coerenti con i filtri selezionati.

Verificare che il layout della stampa sia coerente con il criterio di **Ordinamento** scelto, perché il layout espositivo cambia in base all'ordinamento impostato.

Se l'estrazione è stata generata in Excel, verificare che siano presenti le colonne necessarie per le successive elaborazioni.
