# Tab Note della riga offerta

## Campi
### Tipo nota
Consente di scegliere il tipo di nota da inserire sulla riga.

### Argomento
Consente di richiamare una dicitura fissa, se prevista.

### Note manuali
Consente di scrivere note libere a livello di riga.

## Regole
### Differenza rispetto alla testata
Le note di riga possono essere diverse rispetto alle note inserite nella testata dell'offerta.

### Modalità di compilazione
Per le note di riga è possibile:
- selezionare un tipo nota;
- richiamare un argomento con dicitura fissa;
- inserire manualmente un testo libero.

### Riporto in stampa
Le note di riga possono essere riportate nel dettaglio di stampa dell'offerta in base a come sono state impostate le stampe.
