# Tabella Modalità IVA rimborsi percipienti

## Regole
### Identificazione dei rimborsi
La tabella consente di indicare quali sono le modalità IVA che identificano i rimborsi.

### Valore del rimborso nella registrazione
Le modalità IVA configurate in questa tabella consentono di identificare il valore del rimborso all'interno della registrazione.

### Registrazioni interessate
L'identificazione del valore del rimborso avviene nelle registrazioni di fattura proforma o di fattura passiva.

### Riporto sui dati del compenso
Il valore del rimborso identificato tramite le modalità IVA configurate viene riportato sui dati del compenso.
