# TP Note della riga fattura

## Campi

### Tipo nota
Identifica la tipologia della nota.

Può essere selezionato con il tasto destro del mouse dall'elenco disponibile.

### Argomento
Permette di richiamare una nota già codificata o automatica.

Può essere selezionato con il tasto destro del mouse.

### Note
Campo libero nel quale l'utente può scrivere manualmente il testo della nota.

## Regole

### Inserimento note manuali
Le note di riga possono essere inserite manualmente nel campo note.

### Richiamo note automatiche
Le note possono essere richiamate tramite argomenti creati precedentemente con delle descrizioni fisse.

### Aggiunta righe nota
Le righe nota possono essere aggiunte tramite il pulsante "+" verde.

### Cancellazione righe nota
Le righe nota possono essere cancellate tramite la relativa icona di cancellazione.

### Stampa note
Le note di riga possono comparire in stampa in base alle regole definite nel report di stampa richiesto dal cliente.
