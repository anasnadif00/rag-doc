# Banche

## Campi
### Codice Magia 90
Il campo **Codice Magia 90** è un codice univoco mantenuto per compatibilità con la versione precedente del software Magia.

### Descrizione
Il campo **Descrizione** identifica la banca.

### Codice ABI
Il campo **Codice ABI** contiene il codice ABI della banca.

### Codice CAB
Il campo **Codice CAB** contiene il codice CAB della banca.

### Codice IBAN
Nel campo **Codice IBAN** è possibile inserire l'IBAN della propria società riferito alla banca in oggetto.

## Regole
### Funzione della tabella
La tabella **Banche** contiene l'elenco di tutte le banche utilizzate dall'azienda e/o da soggetti terzi con cui l'azienda è in relazione.
