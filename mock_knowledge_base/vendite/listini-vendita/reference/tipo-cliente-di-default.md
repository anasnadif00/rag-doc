# Tipo cliente di default

## Regole

### Gerarchia clienti proposta di default
La tabella Tipo cliente di default consente di indicare quale gerarchia clienti deve essere proposta come default.

La gerarchia deve essere stata creata tramite Gerarchia clienti nella funzione Gerarchia sconti.

### Utilizzo nella Gerarchia sconti
Il valore configurato viene utilizzato come proposta predefinita nella gestione della Gerarchia sconti.
