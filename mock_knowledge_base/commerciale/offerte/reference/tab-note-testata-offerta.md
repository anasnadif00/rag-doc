# Campi del tab TP Note di testata dell'offerta

## Campi

### Tipo di nota
Consente di selezionare la tipologia della nota da inserire.

### Argomento
Consente di richiamare argomenti preconfigurati da utilizzare come base per la compilazione della nota.

### Nota
Contiene il testo della nota da associare all'offerta.

## Regole

### Tipologie di nota
Le note possono essere di tipo diverso, ad esempio:
1. nota iniziali di testata;
2. nota finale.

### Gestione tramite argomenti
Le note possono essere gestite tramite **Argomenti** preconfigurati in apposita tabella, in modo da proporre formule predefinite di testi usati frequentemente.

### Stampa delle note
Le note possono essere stampate nell'offerta nella posizione scelta dall'utente, in base alla configurazione e alla tipologia utilizzata.
