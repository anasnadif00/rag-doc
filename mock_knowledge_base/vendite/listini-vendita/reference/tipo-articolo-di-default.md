# Tipo articolo di default

## Regole

### Gerarchia articoli proposta di default
La tabella Tipo articolo di default consente di indicare quale gerarchia deve essere proposta come default.

La gerarchia viene creata nel tipo Gerarchia articoli.

### Utilizzo nella Gerarchia sconti
Il valore configurato viene utilizzato nella funzione Gerarchia sconti come gerarchia articoli proposta di default.
