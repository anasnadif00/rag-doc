# Campi e regole del TP Note

## Regole

### Origine delle note
Nel TP Note vengono visualizzate le note provenienti da documenti precedenti che hanno generato la fattura.

### Inserimento manuale
È possibile inserire nuove note manualmente.

### Utilizzo degli argomenti
Le note possono essere richiamate utilizzando gli Argomenti preconfigurati.

### Note del vettore
Se il vettore associato alla fattura contiene note configurate nella relativa tabella, tali note vengono riportate automaticamente.

Le note riportate possono essere modificate dall'utente.

### Gestione in stampa
La stampa e la visualizzazione delle note dipendono:

- dalle caratteristiche del tipo nota utilizzato
- dalle impostazioni del layout di stampa della fattura
