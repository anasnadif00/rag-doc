# Tab Dati di analitica della riga offerta

## Campi
### Da data competenza
Indica la data iniziale di competenza analitica della riga.

### A data competenza
Indica la data finale di competenza analitica della riga.

### CDC
Indica il centro di costo associato alla riga.

## Regole
### Funzione del tab
Il tab **Dati di analitica** gestisce le informazioni di contabilità analitica che verranno riprese nei passaggi successivi dell'offerta, come ordine, DDT o fatturazione.

### Riporto dalla testata
Se i dati di analitica sono valorizzati in testata, vengono normalmente riportati automaticamente anche sulle righe.

### Modificabilità a livello riga
I dati riportati dalla testata possono comunque essere modificati riga per riga.
