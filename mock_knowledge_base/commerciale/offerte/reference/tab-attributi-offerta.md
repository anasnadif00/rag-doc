# Campi del tab TP Attributi dell'offerta

## Campi

### Attributi
Consentono di specificare caratteristiche aggiuntive relative all'offerta.

## Regole

### Configurazione preventiva
Le caratteristiche aggiuntive disponibili nel tab devono essere precedentemente caricate nella tabella **Caratteristiche attributi**.
