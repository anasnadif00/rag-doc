# Stampa registro DDT conto visione

## Campi
### Filtri disponibili
La stampa consente di filtrare per:
- cliente
- tipo DDT
- intervallo numeri DDT
- intervallo date
- stato DDT
- intervallo date scadenza
- agente
- articolo

### Flag disponibili
È possibile includere:
- reso
- guasto
- fatturato

## Regole
### Stato DDT
Lo stato del DDT può essere:
- aperto
- chiuso

### Informazioni riportate
La stampa restituisce:
- cliente destinatario
- numero DDT
- data DDT
- riferimenti documento
- valore DDT
- importi fatturati
- destinazione merce
- articoli presenti nel DDT
