# Tabella Argomenti clienti

## Regole

### Utilizzo degli argomenti clienti
La tabella "Argomenti clienti" consente di creare note pre-codificate da agganciare alle note inserite negli ordini clienti.

L'argomento clienti permette quindi di predisporre testi standard che possono essere richiamati nelle note dell'ordine.

### Creazione di un argomento cliente
Per creare un nuovo argomento cliente si utilizza il tasto più verde.

Dopo la creazione, l'argomento può essere agganciato alle note dell'ordine. Le note definite nell'argomento vengono riportate nelle note dell'ordine quando l'argomento viene selezionato.

## Campi

### Codice
Identifica l'argomento cliente.

Il codice è scelto dall'utente.

### Descrizione
Descrive l'argomento cliente creato 

### Note
Contiene il testo che dovrà essere riportato nelle note dell'ordine quando l'argomento cliente viene agganciato.
