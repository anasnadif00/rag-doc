# Pannello Clienti di spedizione della gestione Conti

## Campi
### Conto
Consente di indicare il conto del cliente di spedizione. Il valore può essere inserito manualmente oppure richiamato tramite Help.

### Default
Consente di specificare il cliente di spedizione da utilizzare come predefinito.

### Ragione sociale
Visualizza la ragione sociale già registrata in Magia per il cliente di spedizione selezionato.

### Codice di avviamento postale
Visualizza il codice di avviamento postale già registrato in Magia per il cliente di spedizione selezionato.

### Via
Visualizza la via già registrata in Magia per il cliente di spedizione selezionato.

### Città
Visualizza la città già registrata in Magia per il cliente di spedizione selezionato.

### Provincia
Visualizza la provincia già registrata in Magia per il cliente di spedizione selezionato.

## Regole
### Ambito del pannello
Attraverso il pannello Clienti di spedizione dell'anagrafica Conti è possibile indicare, per i conti di tipo cliente, uno o più clienti di spedizione.

### Cliente di spedizione predefinito
Tra i clienti di spedizione indicati è possibile specificarne uno di default.

### Utilizzo nei moduli
I clienti di spedizione definiti nel pannello vengono utilizzati dai moduli del commerciale e dal modulo fatturazione per l'emissione dei documenti.

### Compilazione dei dati
Dopo avere indicato il conto del cliente di spedizione, vengono visualizzati i dati anagrafici già presenti in Magia.
