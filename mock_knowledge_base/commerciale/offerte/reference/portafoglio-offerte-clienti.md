# Portafoglio offerte clienti

## Regole
### Finalità della stampa
La stampa **Portafoglio offerte clienti** consente di estrarre una situazione delle offerte in base ai filtri impostati dall'utente. Vengono estratte le informazioni relative alla testata delle offerte, gli articoli inseriti, la data di scadenza dell'offerta, le quantità inserite ed eventualmente quelle residue. La data di avanzamento, il motivo e la probabilità di chiusura se gestite nell'offerta.

### Filtri
La stampa è gestita tramite una serie di filtri impostabili dall'utente per delimitare le offerte da analizzare.

### Ordinamenti disponibili
L'estrazione può essere ordinata secondo il criterio scelto dall'utente:
- per **Cliente**
- per **Ragione Sociale**
- per **Offerta**
- per **Articolo**
