# TP Note del DDT

## Campi

### Note
Contiene le note associate al DDT.

### Argomento
Consente di richiamare note predefinite per agevolare l'inserimento.

### Vettore
Se è stato specificato un vettore con note associate, le note vengono proposte automaticamente.

## Regole

### Inserimento manuale
Le note possono essere inserite manualmente.

### Inserimento tramite argomento
Le note possono essere richiamate tramite argomento usando l'icona di ricerca.

### Note da vettore
Se al vettore selezionato sono associate note nella relativa tabella, queste vengono proposte automaticamente.

Le note proposte possono essere modificate.

### Gestione e stampa
Le note vengono gestite e stampate in base alle caratteristiche del tipo nota utilizzato.
