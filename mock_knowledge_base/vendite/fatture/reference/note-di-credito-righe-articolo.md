# Note di credito con righe articolo negative

## Regole

### Tipo documento
Per inserire una nota di credito deve essere utilizzato un tipo documento che identifichi la nota di credito e non la fattura.

### Quantità negativa
La quantità deve essere inserita con segno meno.

### Prezzo unitario negativo
Il prezzo unitario deve essere inserito con segno meno.

### Documento negativo
Gli importi devono essere espressi in negativo per generare correttamente il documento di nota di credito.
