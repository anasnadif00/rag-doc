# TP Fatturazione elettronica ed esito SDI

## Campi

### Fase avanzamento lavori

Consente di gestire le informazioni richieste per la fatturazione verso la Pubblica Amministrazione.

### Fattura di anticipo

Permette di identificare fatture emesse a titolo di anticipo.

### Fatture collegate

Consente di collegare altri documenti, ad esempio nel caso di note di credito.

### Dichiarazione d'intento

Se presente vengono visualizzati:

- numero di protocollo;
- progressivo;
- data di attivazione.

## Campi del TP Esito SDI

### Id SDI

Identificativo assegnato dal Sistema di Interscambio.

### Data inserimento

Data di registrazione delle informazioni SDI.

### Data invio SDI

Data di trasmissione del documento.

### Nome file

Nome del file XML generato.

### Stato fatturazione elettronica

Riporta lo stato del processo e gli eventuali errori restituiti.

## Regole

### Consultazione XML

Mediante il pulsante con il simbolo dell'Agenzia delle Entrate è possibile:

- visualizzare il file XML;
- visualizzarne le rappresentazioni di stampa;
- salvare il file XML;
- scaricare il file XML.
